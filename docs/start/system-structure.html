# 架构设计

所有服务基于Docker，通过容器技术承载各类服务，原创Trojan Panel Core，适配各种代理软件。这是占内存最小而且不需要自己手动申请/续签证书的轻量级方案。实测，把所有服务器都部署完，1H1G的服务器足够用。
